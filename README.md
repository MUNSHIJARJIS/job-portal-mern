# job-portal-mern
A simple job portal application built with the MERN stack (MongoDB, Express.js, React, Node.js) that allows:

User registration and login (both job seekers and employers)

Job listings browsing

Job posting functionality for employers
Technologies Used
Frontend
React.js

React Router for navigation

Axios for API calls


Backend
Node.js

Express.js

MongoDB (with Mongoose ODM)

JWT for authentication

Bcrypt for password hashing

Installation
Prerequisites
Node.js (v14 or later)

MongoDB (local or Atlas cluster)

npm or yarn

stup backend

cd backend
npm install

setup frontend
cd ../frontend
npm install

cd backend
npm start

cd frontend
npm start
